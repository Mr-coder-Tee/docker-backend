-----------------------------Step to run backend locally------------------------------------
step 1: install docker on your PC. Link:https://docs.docker.com/desktop/install/windows-install ( use youtube if confused by steps in the link)
step 2: once docker is properly installed create a network by opening CMD or gitbash paste : 
 i) docker network create --driver bridge local-diskie-net
ii) docker volume create local-diskie-data
